Automated Verification & Integrity Device

Project Overview

AVID Journal is an automated framework designed to assist in the peer-review process of mathematical literature. It represents a novel approach to "Auto-formalization", aiming to bridge the gap between unstructured LaTeX documents and formal verification systems.

The core challenge addressed by this project is the "Hallucination Problem" in LLMs when dealing with strict logical syntax. By implementing a guided translation layer, AVID ensures that the generated code is not only syntactically correct but semantically faithful to the original theorem.

> ./view --pipeline

  • 1. Ingestion Raw .tex file processing and metadata extraction. We use custom parsers to strip non-semantic LaTeX macros.
  • 2. Structural Parsing Semantic segmentation. Identifying Boundaries of Definitions, Theorems, Lemmas, and Proofs using a finetuned BERT model.
  • 3. Formal Translation Neural-driven mapping. A Specialized LLM (based on CodeLlama) translates natural language math into Coq or Lean 4 code.
  • 4. Formal Verification The generated code is passed to the Rocq (Coq) prover. If the proof checks, the theorem is marked as "Verified".
  • 5. Novelty Evaluation Embedding-based redundancy detection. We compare the vectorized theorem statement against a vector database of the Mathlib library.
  • 6. Publication Generation of verification metrics and formal validation reports for human reviewers.
